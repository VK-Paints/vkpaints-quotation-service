# VK-Paints Quotation Service

## Description
Calculates price estimates and generates paint quotations based on project dimensions.

## Tech Stack
- Node.js & Express
- PostgreSQL & Sequelize

## Environment Variables
- PORT: Service port (default 3003)
- DB_URL: PostgreSQL connection string

## API Endpoints
- POST /api/quotation: Calculate quote
- GET /health: Health status
